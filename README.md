# Chemotherapy Toxicity Predictor

Chemotherapy can be life-saving — but it can also come with serious side effects.

This project uses machine learning to **predict the risk of chemotherapy toxicity** based on patient data, helping doctors make safer and more informed treatment decisions.

---

## What it does

- Takes patient details (age, BMI, symptoms, etc.)
- Uses a trained ML model to analyze risk
- Gives:
  - Toxicity risk score
  -  High / Low risk prediction
  -  Key contributing factors

---

##  Why this matters

Chemotherapy toxicity can be unpredictable and dangerous.

Early prediction can:
- Prevent complications
- Help adjust treatment plans
- Improve patient safety

---

## 🛠 Tech Stack

- **Backend:** Flask (Python)
- **Model:** Random Forest (Scikit-learn)
- **Frontend:** HTML, CSS, JavaScript
